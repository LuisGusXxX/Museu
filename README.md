# Museu
Trabalho unip 

este código em C é um sistema simples de gerenciamento de vendas de ingressos para um museu multitemático. Ele permite selecionar temas do museu, comprar ingressos, exibir informações sobre vendas e formas de pagamento, além de gerar uma nota fiscal em um arquivo CSV.

Resumidamente, as principais funcionalidades são:

Selecionar Tema do Museu: Permite ao usuário escolher até quatro temas diferentes para visitar no museu.
Compra de Ingressos: Oferece a opção de comprar ingressos de diferentes tipos (inteiro, meia ou gratuito) e acompanha a contagem dos ingressos vendidos e o valor total arrecadado.
Exibir Total de Ingressos Vendidos e Valor Arrecadado: Mostra os totais de ingressos vendidos e o montante arrecadado até o momento.
Formas de Pagamento: Apresenta opções de formas de pagamento para o cliente escolher.
Geração de Nota Fiscal: Exibe uma nota fiscal com detalhes das compras realizadas, formas de pagamento escolhidas e gera um arquivo CSV com essas informações.
As funções são organizadas de forma a permitir a interação do usuário, coletar dados das vendas, mostrar informações relevantes e armazenar esses dados para fins de registro.
